About Me

I'm a software engineer with experience building web applications using Ruby on Rails, JavaScript, React, Redux, Express.js, Mongoose, MongoDB, HTML, CSS, and many other tools. I've also started coding in Python recently and absolutely love learning new technologies.

I recently graduated from the software engineering curriculum at App Academy, a highly selective and immersive software engineering with a ~3% acceptance rate. Here, I spent over 100 hours a week honing my coding skills and building projects like Mocdoc. This is a project that encapsulates my passions in empowering patients and coding.
